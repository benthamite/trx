#+title: trx
#+subtitle: Emacs interface for the Transmission BitTorrent client
#+author: Mark Oteiza, Pablo Stafforini

The comprehensive manual for =trx.el= is in [[file:../README.org][README.org]] at the repository root.

The ERT test suite (168 tests) is in =trx-test.el=.  Run it with =make test=.

Jackett search integration is provided by =trx-jackett.el= (requires a running Jackett instance and =curl=).  The API key is auto-discovered from Jackett's =ServerConfig.json=.  Per-column faces and fade-out truncation apply to all tabulated-list buffers.  The torrent list auto-refreshes after remove and delete operations.
